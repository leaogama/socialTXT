    O SocialTXT implementa a especificação Model Context Protocol (MCP). Conecte-o diretamente como uma ferramenta nativa ao Claude Desktop, Cursor ou qualquer cliente compatível.

    Passo 1: Adicione esta configuração ao seu arquivo de configuração MCP.
    No Windows (Claude Desktop): %APPDATA%\Claude\claude_desktop_config.json 

    {
  "mcpServers": {
    "socialtxt": {
      "command": "python",
      "args": ["C:/_git/socialTXT/mcp_server.py"],
      "env": {
        "LLM_API_KEY": "SUA_API_KEY_AQUI"
      }
    }
  }
}

    Passo 2: Certifique-se de que a biblioteca oficial do MCP está instalada no seu Python local:
    pip install mcp
